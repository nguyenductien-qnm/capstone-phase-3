QUANTITATIVE ENDPOINT DECISION
- S3 Gateway Endpoint: no additional hourly or data-processing charge; selected.
- ECR interface endpoints considered: ecr.api + ecr.dkr.
- HA scope is three AZs, so fixed footprint would be 2 endpoints x 3 AZ x 730 h x $0.01/endpoint-AZ-hour = approximately $43.80/month, plus $0.01/GB PrivateLink processing for the first PB.
- NAT remains for Internet egress. Avoided processing margin is approximately $0.045 NAT/GB - $0.01 endpoint/GB = $0.035/GB.
- Approximate interface break-even: $43.80 / $0.035 = 1251 GB/month of attributable ECR interface traffic, before minor cross-AZ effects.
- Current account-wide NAT bytes extrapolate to about 232.4 GB/month (61.9773 GB / 8 days x 30), and only a subset is ECR API/registry traffic.
- Decision: do not add ecr.api/ecr.dkr, Secrets Manager, STS, EC2, SSM or other interface endpoints yet.
